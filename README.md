# FitTrack Fitness Tracking API

Full-featured REST API for tracking workouts, meals, and fitness goals.

## Features
- 4 MongoDB collections (Users, Workouts with 8+ fields, Meals, Goals)
- GitHub OAuth authentication
- JWT protected routes
- Full CRUD + data validation + error handling
- Professional Swagger docs at `/api-docs`
- Deployed on Render

Live: https://final-project-27de.onrender.com/api-docs
